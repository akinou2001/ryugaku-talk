# Phase 10: 検証・チェック

データベースの状態を確認するSQLファイルです。実行は任意です。

## 📋 ファイル一覧

- **`check-rls-policies.sql`** - RLSポリシーの確認

## 🚀 実行方法

これらのSQLファイルは、データベースの状態を確認するために使用します。

- マイグレーション後に、正しく設定されているか確認する場合に実行

- 定期的なメンテナンス時に実行

## ⚠️ 注意事項

- これらのSQLファイルは**読み取り専用**です（データを変更しません）
- いつでも安全に実行できます
- エラーが発生した場合は、データベースの状態を確認してください
