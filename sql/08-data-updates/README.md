# Phase 8: データ更新

既存データの更新です。**本番環境では通常実行不要**です。

## 📋 ファイル一覧

- **`update-japanese-names.sql`** - 日本語名の更新
- **`update-japanese-names-from-csv.sql`** - CSVからの日本語名更新
- **`remove-duplicate-universities.sql`** - 重複大学の削除
- **`reset-japanese-names.sql`** - 日本語名のリセット

## ⚠️ 重要な注意事項

### 本番環境での実行

- **新規環境への適用時は通常実行不要**です
- 既存データを更新するSQLなので、新規環境ではデータが存在しません

### 既存環境での実行

- 既存の本番環境でデータを更新する場合のみ実行してください
- **実行前に必ずバックアップを取得**してください
- データ更新は不可逆的な操作の可能性があるため、慎重に実行してください

## 🚀 実行方法

必要に応じて、個別にSQLファイルを実行してください。すべてを実行する必要はありません。
