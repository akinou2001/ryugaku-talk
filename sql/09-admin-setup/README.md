# Phase 9: 管理者アカウント設定

管理者アカウントの作成です。**本番環境では必要に応じて実行**してください。

## 📋 ファイル一覧

- **`setup-admin-account.sql`** - 管理者アカウントの作成
- **`setup-new-admin.sql`** - 新しい管理者の作成
- **`delete-admin-user.sql`** - 管理者ユーザーの削除（注意）

## ⚠️ 重要な注意事項

### 実行タイミング

- 本番環境に初めてデプロイする際に、管理者アカウントを作成する場合に実行
- 新しい管理者を追加する場合に実行

### セキュリティ

- 管理者アカウントは強力な権限を持つため、慎重に管理してください
- `delete-admin-user.sql`は管理者を削除するため、実行前に十分確認してください

## 🚀 実行方法

1. **管理者アカウントを作成する場合**
   - `setup-admin-account.sql` または `setup-new-admin.sql` を実行
   - SQLファイル内のメールアドレスやユーザーIDを適切に設定してください

2. **管理者を削除する場合**
   - `delete-admin-user.sql` を実行（注意が必要）

## 📝 カスタマイズ

各SQLファイルを開いて、メールアドレスやユーザーIDを適切に設定してください。
