# RyugakuTalk プロジェクト概要（共同開発者向け）

## 📌 プロジェクトの概要

**RyugakuTalk**は、留学中・留学希望者・関係者が質問・共有・交流できるオンラインコミュニティプラットフォームです。

### 目的
- 留学に関する情報共有
- 留学経験者と希望者のマッチング
- 安全なコミュニティ形成

---

## ✅ 現状の進捗

### 実装完了機能
✅ ユーザー登録・ログイン  
✅ プロフィール管理  
✅ 掲示板機能（投稿・コメント・いいね）  
✅ チャット機能（DM）  
✅ 検索・フィルタリング  
✅ 多言語対応（日本語・英語）

### 技術スタック
- **フロントエンド**: Next.js 14 + TypeScript + Tailwind CSS
- **データベース**: PostgreSQL（Supabase経由）
- **認証**: Supabase Auth
- **デプロイ**: Vercel（現在）

### リポジトリ
- **GitHub**: https://github.com/akinou2001/ryugaku-talk
- **ブランチ**: main
